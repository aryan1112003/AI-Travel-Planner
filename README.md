## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file and add your Gemini API key:
   ```
   VITE_BARD_API_KEY=your_api_key_here
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## Usage

1. Search for your desired destination
2. Set your travel budget and preferred currency
3. Select travel dates
4. Generate AI-powered itinerary
5. Download your personalized travel plan
